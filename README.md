
Проект представляет собой реализацию простой игры в терминале. Проект - прототип игры в лабиринт, цель игры собрать все монетки, не наткнуться на стены.

Игровое поле представляет собой сетку ячеек, в которых находятся элементы для взаимодействия игрока.

Игрок (класс "Player") может перемещаться по полю, используя клавиши на клавиатуре. Он также может взаимодействовать с элементами на поле, такими как монеты или другие объекты игры. Каждое взаимодействие игрока с элементом может повлиять на его очки и жизни, а также изменить состояние самого элемента.

Цель проекта - создание простой игры для развлечения пользователя. Она позволяет пользователю управлять персонажем внутри терминала и взаимодействовать с разными объектами на игровом поле.

Запуск проекта:

g++ -c player.h

g++ -c field.h

g++ -c imaging.h

g++ -c -no-pie player.cpp field.cpp imaing.cpp main.cpp

g++ main.o -o output -lncurses
./output
